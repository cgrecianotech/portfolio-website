What is Cloud Computing?

Ref: https://learn.cantrill.io/courses/2022818/lectures/45718076

  • Summary

    Cloud computing is a model for enabling ubiquitous, convenient, on-demand network access to a shared pool of configurable computing resources (e.g., networks, servers, storage, applications, and services) that can be rapidly provisioned and released with minimal management effort or service provider interaction. This is according to the widely used definition provided by the National Institute of Standards and Technology (NIST) in publication 800-145. The concept of cloud computing has revolutionized the way businesses and individuals access and use computing resources, as it allows for greater flexibility and scalability while reducing the need for costly physical infrastructure and maintenance.

Cloud Computing

  • 🔧 Def: Model for enabling ubiquitous, convenient, on-demand NW access to a shared pool of configurable computing resources (e.g. NWs, servers, storage, apps, services)
    • Computing resources can be rapidly provisioned & released with minimal management effort or service provider interaction
  • Cloud computing revolutionized the way businesses and individuals access and use computing resources
    • Advantages: ↑ scalability, ↑ flexibility, ↓ maintenance, ↓ physical infrastructure

Five Characteristics of a Cloud Platform

  • 🔧 Five characteristics are:
    • On-Demand Self-Service
    • Broad NW Access
    • Resource Pooling
    • Rapid Elasticity
    • Measured Service
    • Summary

ON-DEMAND SELF-SERVICE

  • Customers can provision resources as needed without human interaction
    • 💡 Management can be done through UI, CLI or APIs

BROAD NETWORK ACCESS

  • Resources are available over the NW and accessed through standard mechanisms
    • 💡 No need to use niche devices nor non-standard NW protocols

RESOURCE POOLING

  • Resources are pooled to serve multiple consumers using a multi-tenant model
  • Sense of location independence → no control/knowledge of exact location of resources
  • ❗ Consumers benefit from economies of scale
    • 💡Cheaper to buy a lot of infrastructure/hardware in bulk. Cloud providers can offer discounted prices for their HW to consumers.

RAPID ELASTICITY

  • Resources can be elastically provisioned and released to scale rapidly outward/inward with demand
    • Never overprovision → no wasted money
    • Never underprovision → no performance issues
  • To the consumer, the capabilities available for provisioning often appear to be unlimited
  • ‼️ Most important benefit of cloud computing!!

MEASURED SERVICE

  • Resource usage can be monitored, controlled, reported… and most importantly, BILLED
  • With correct configuration, you only pay for what you consume (pay-as-you-go)

Six Advantages of Cloud Computing (AWS whitepaper)

Ref: https://docs.aws.amazon.com/whitepapers/latest/aws-overview/six-advantages-of-cloud-computing.html

  1. Trade fixed/capital expense (CAPEX) for variable/operational expense (OPEX)
    • Reduced Total Cost of Ownership (TCO) & Operational Expense (OPEX)
    • Pay on-demand: customers don’t own hardware
  1. Benefit from massive economies of scale
    • Prices are reduced as AWS is more efficient due to large scale
  1. Stop guessing capacity
    • Scale based on actual measured usage (no idle resources)
  1. Increase speed and agility
    • Resources can be provisioned/released much faster than when owning actual hardware
    • Cost and time to experiment and develop is much lower
  1. Stop spending money running and maintaining data centers
    • Reduced admin overhead for infrastructure, focus on your own customers
  1. Go global in minutes
    • Easily deploy in multiple regions with just a few clicks
    • Leverage the AWS global infrastructure → lower latency and better UX
