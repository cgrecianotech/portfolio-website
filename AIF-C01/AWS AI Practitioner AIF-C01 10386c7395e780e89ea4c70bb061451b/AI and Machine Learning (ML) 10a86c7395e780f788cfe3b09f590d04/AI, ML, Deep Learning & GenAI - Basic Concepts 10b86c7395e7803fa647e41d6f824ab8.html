Artificial Intelligence (AI)

  • 🔧 Broad field for the development of intelligent systems capable of performing tasks that typically require human intelligence:
    • Perception, Reasoning, Learning, Problem solving, Decision-making…
    • Umbrella-term for various techniques
  • Diagram

Machine Learning (ML)

  • 🔧 Subset of AI whereby lots of data (training data) is fed into an algorithm, which outputs a statistical model and rules
    • ❗No explicit programming of rules → rules “learned” from training data
    • ML systems can make predictions based on patterns of their training data
  • Examples: Regression and Classification algorithms
    • 💡 In both systems, the algorithm can make estimated guesses for samples not provided in training data
  • ‼️ AI ≠ ML!!
    • Example: MYCIN Expert System from 1970s is AI with hundreds of predefined rules to determine culprit bacteria
    • A set of if-else statements could be considered barebones AI

When is Machine Learning NOT appropriate?

  • Deterministic problems (where the solution can be computed) are better solved with computer code adapted to the problem than with ML
    • ML algorithms can offer an approximation of the result, but they're always worse and less optimal than a direct coding solution →
  • Example: Probability of drawing a blue card in a deck of cards

Deep Learning (DL)

  • 🔧 Subset of ML which involves applying several “learning layers” to the training data, which form a neural network
    • DL can process and detect more complex patterns than traditional ML
    • Examples:
      • Computer Vision → images
      • Natural Language Processing (NLP) → text
    • Diagram of a neural network
  • Requires even more input data than traditional ML
  • Requires Graphical Processing Units (GPUs), which excel at highly parallel computing
    • 💡 Nvidia currently making record profits due to their GPUs

Neural Networks

  • 🔧 Neurons (nodes) and synapses (connections)
    • 💡 look similar to human brain
  • Neurons and synapses updated by DL algorithms (very complicated math) as they process more and more training data
    • Neural networks may have billions of nodes
  • Classic example of neural network: recognize handwritten digits

Generative AI (GenAI)

  • Intro:
  • 🔧 Subset of DL, focused on generating new data from lots of training data
    • Multi-purpose FMs backed by DL neural networks (text generation neural networks, text summarization neural networks…)
    • Diagram

GenAI Components

  • Data Layer – collect vast amounts of data
  • ML Framework and Algorithm Layer – data scientists and engineers work together to understand use cases, requirements, and frameworks that can solve them
  • Model Layer – implement a model and train it, we have the structure, the parameters and functions, optimizer function
  • Application Layer – how to serve the model, and its capabilities for your users

Transformer Model (LLM)

  • Able to process a sentence as a whole (instead of word by word)
    • Faster and more efficient text processing (less training time)
    • Self-attention mechanisms and encoders give relative importance/weight to specific words in a sentence → more coherent sentences
  • Examples: Google BERT, OpenAI ChatGPT
    • GPT = Generative Pre-trained Transformer
  • Transformer-based LLM Diagram

Multi-modal Models

  • Capable of handling multiple types of input (text, image, audio…) and/or generating multiple types of output (text, video…)
  • Example: “generate a video making the picture of the cat speak the audio that is included”

Human thinking can be AI-like in different scenarios

  • AI: Sometimes we know “if this happens, then do that”
    • 💡 e.g. “if it's raining, take an umbrella”
  • ML: Sometimes we’ve seen a lot of similar things before, and we classify them
    • 💡 e.g. “I have seen 1000 dogs, when I see a new dog, I'm confident it's a dog because it follows patterns”
  • DL: Sometimes we haven’t seen something before, but we have
    “learned” a lot of similar concepts, so we can make a decision
    • 💡 e.g. “I have seen cats, dogs, elephants, horses… I now see a tiger for the first time. It doesn't look like any of the animals I’ve seen before… but I do know it's an animal because of similar patterns”
  • GenAI: Sometimes, we get creative, and based on what we’ve learned, we can generate content
    • 💡 e.g. “I invent a new animal: the griffin. It's half-eagle, half-lion”
